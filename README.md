# BlogExpress

## Description

BlogExpress is a mobile application that allows users to read and interact with blog posts on their Android devices. It provides a convenient and user-friendly interface for browsing articles, commenting on posts, and staying updated with the latest content from the blog.

## Features
-**Login/SignUp:** Users can Sign up and login on the platform.
- **Browse Articles:** Users can explore a list of blog posts with titles and brief summaries, making it easy to find articles of interest.
- **Read Full Articles:** Tapping on a post opens the full article in a clean and readable format, ensuring an enjoyable reading experience.
- **Commenting:** Users can leave comments on articles to share their thoughts or engage in discussions with other readers.
- **Notifications:** The application sends push notifications to users whenever a new blog post is published, ensuring they never miss any updates.
- **Bookmarking:** Users can save articles they want to revisit later by bookmarking them, allowing quick access to their favourite content.
- **Create Blog** Users can create a blog and can share it with the platform.

## Getting Started

To run the My Blog Android Application locally, follow these steps:

1. Clone the repository: `git clone https://github.com/adityait019/BlogExpress`
2. Open the project in Android Studio.
3. Build and run the application on an emulator or a connected Android device.
4. Start exploring and interacting with the blog posts.

## Technologies Used

- Android Studio
- Java
- XML
- RESTful APIs (for fetching blog post data)
- Firebase Authentication
- Firebase Cloud Messaging (for push notifications)
- SQLite (for local data storage)
